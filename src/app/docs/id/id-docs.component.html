The ID directive assigns a unique ID to an element and provides that ID to other elements.

Use the ID directive when one HTML element must be associated with another HTML element by its id attribute, such as the label element's for attribute. Use the ID directive to ensure unique IDs do not conflict with other elements on the page.

Don't use the ID directive when you don't need to associate HTML elements.

Don't use the ID directive in combination with hard-coded IDs because this can cause collisions. We recommend only using the ID directive.

The ID module provides a directive that assigns a unique ID to an element and provides that ID to other elements.

Use the ID directive when one HTML element must be associated with another HTML element by its id attribute, such as the label element's for attribute. To prevent errors with functionality and accessibiliity, use the ID directive to guarantee unique IDs will not conflict with other elements on the page.

Don't use the ID directive when an element does not need to be referenced by its ID attribute.

Don't use the ID directive to hard-code an ID to element, which can lead to ID collisions.
